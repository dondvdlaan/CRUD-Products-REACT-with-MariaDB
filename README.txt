CRUD of Products with REACT and MariaDB

Simple CRUD(Create, Read, Update and Delete) of standard products with REACT in Front-End and Nodejs and MariaDB in Back-End.

REACT will handle the GUI and CRUD actions, AXIOS is used to communicate with the Nodejs server, where express is used to Mysql with MariaDB.

Summary:
- REACT
- NOdejs
- MariaDB
- Ubuntu OS
- Bootstrap for CSS
